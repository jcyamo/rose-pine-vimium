## Usage

1. Open Vimium options (click the extension icon → Options, or navigate to `chrome-extension://dbepggeogbaibhgnhhndojpepiihcmeb/pages/options.html`)
2. Scroll to **Custom CSS for Vimium UI**
3. Paste the contents of [`vimium.css`](vimium.css)

The theme auto-switches between **Rosé Pine Moon** (dark) and **Rosé Pine Dawn** (light) based on your system's color scheme preference. To use **Rosé Pine** (main) for dark mode instead, swap the active/inactive variable blocks at the top of the file.
